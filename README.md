# Peek

웹 페이지의 UI 요소를 검사하고 LLM 프롬프트용 컨텍스트를 복사하는 Chrome 확장입니다.

## 설치

1. Chrome에서 `chrome://extensions` 접속
2. 오른쪽 상단 **개발자 모드** 활성화
3. **압축해제된 확장 프로그램을 로드합니다** 클릭 → 이 폴더 선택

## 사용법

1. 원하는 웹 페이지에서 **Peek** 패널의 `Start` 클릭
2. 마우스를 올려 요소 하이라이트 확인
3. 클릭하여 요소 잠금
4. **Copy** 버튼으로 컨텍스트 복사

### 단축키

| 키 | 동작 |
|---|---|
| `Cmd/Ctrl + ↑` | 부모 요소로 이동 |
| `Cmd/Ctrl + ↓` | 첫 번째 자식 요소로 이동 |
| `Cmd/Ctrl + ←/→` | 형제 요소로 이동 |
| `Esc` | 잠금 해제 |

## 복사되는 정보

- DOM 경로
- 주요 스타일
- 데이터 속성 및 ARIA 속성
- React 컴포넌트 소스 위치 (개발 모드 한정)
- HTML 스니펫
